## Overview

This repository contains my solution for **Assignment 2** of the **SE302T - Database Management Systems** course. The assignment focuses on recreating modern web interfaces using HTML and CSS, generating dynamic content with JavaScript, and implementing basic form validation.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Tasks

### Task 1
**Facebook Login Page**
- Facebook-inspired login interface
- Built using HTML and CSS only

### Task 2A
**Google Sign In**
- Google Sign In page
- HTML and CSS implementation

### Task 2B
**Google Sign Up**
- Google Create Account page
- HTML and CSS implementation

### Task 3
**Notion Landing Page**
- Navigation Bar
- Hero Section
- Built using HTML and CSS

### Task 4A
**Dynamic Product Cards**
- Product cards generated dynamically using JavaScript
- Each product includes:
  - Name
  - Image
  - Description
  - Price

### Task 4B
**Dynamic Blog Cards**
- Blog cards generated dynamically using JavaScript
- Each blog includes:
  - Title
  - Image
  - Description
  - Date/Tag

### Task 5
**Signup Form Validation**
- Name, Email, Password, and Confirm Password fields
- JavaScript validation for:
  - Password length (minimum 6 characters)
  - Password confirmation
